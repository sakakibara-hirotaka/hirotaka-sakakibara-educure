1-8_環境構築_3（VSCodeとJDK）（2025_01_事前準備）

VSCodeでJavaを使えるように設定を行います。

エラーが起きた場合はそのまま進めず、講師への確認を行いましょう

8. 環境構築3

8-1. 拡張機能

前の章でVSCodeをインストールしていただきましたが、それだけではJavaを使えるようにはなりません。 拡張機能をインストールする必要があります。

拡張機能は、ソフトウェアに新しい機能を追加するための小さなプログラムやプラグインのことです。 VSCodeは、拡張機能を通じて多様な機能を追加できる柔軟なエディタです。これにより、開発者は自分のニーズに合わせてエディタをカスタマイズし、効率的な開発環境を構築できます。

今回はJavaを使えるようにするための拡張機能をインストールします。

  • JDKのインストール（済）
    • JDKのダウンロード
    • JDKのインストール
  • 拡張機能のインストール
  • JDKの設定
  • Javaのプログラムの動作確認
    • Javaのプロジェクトの作成
    • 動作確認用のJavaファイルの作成

この流れでインストールしていきます。説明を読みながら順番に進めてください。

8-2. JDKのインストール

前のセクションで実施したので、割愛します。

8-3. 拡張機能のインストール

これだけではまだJavaを使えるようにはなりません。 次にVSCodeでJavaを使えるようにするための拡張機能をインストールします。

【Windows】 Windowsの検索バーで「VSCode」と入力すると、「Visual Studio Code」が出てくるので、開いてください。

【Mac】 Macでは以下のいずれかの方法でVSCodeを開きます：

  • Launchpadから「Visual Studio Code」をクリックする
  • ApplicationsフォルダからVSCode（Visual Studio Code）をダブルクリックする
  • Spotlight検索（Command + スペース）で「Visual Studio Code」と入力して開く

開いたら、左側のメニューから「拡張機能」を選択します。 画像の赤い四角で囲ったアイコンです。

検索画面が出てくるので、ここに「Java」と入力すると、「Extension Pack for Java」という拡張機能が出てきます。 右下の「Install」をクリックしてインストールしましょう。

インストールには少し時間がかかります。 このように灰色で「Installing」と表示されていれば、インストール中ですのでしばらく待ちましょう。

このように歯車のマークが出ればインストール完了です。

8-4. JDKの設定

では次に、JDKの設定を行います。

まず先ほどの歯車マークをクリックして「Settings」を開きます。

以下のような画面が出てきます。 検索窓に「java」と入力し、少しスクロールすると、「Java: Home」という項目があります。 項目の中にある「Edit in settings.json」をクリックしてください。

このような画面が開きます。
Settings.jsonのタブになっているのを確認し、エディタの中の赤く囲った箇所にJDKを保存したフォルダを入力します。
JDKを保存したフォルダのパスは、先ほどのJDKのインストールの際に保存先のフォルダをスクリーンショットやスマホのカメラで撮っていただいたと思いますので、そちらを参考にしてください。

Windowsの場合: このPCの環境では、C:\\Program Files\\Java\\jdk-21となっています。
インストールしたバージョンによってパスが異なることがありますので、必ずご自分がダウンロードした際のフォルダを入力するようにしてください。

Macの場合: 通常は /Library/Java/JavaVirtualMachines/jdk-21.jdk/Contents/Home のようなパスになります。 バージョンによってパスが異なる場合がありますので、実際のインストール先を確認してください。

入力したら保存しましょう。 【Windows】「Ctrl + S」を押します 【Mac】「Command + S」を押します

上記のコマンド入力後、画像のようにタブの横の●が消えていれば保存できています。

保存されたのを確認したら一度VSCodeを閉じてください。

8-5. Javaのプログラムの動作確認

最後に、Javaのプログラム用のフォルダを作成し、Javaプログラムを作成して動作確認を行います。

Javaのプロジェクトの作成

メニューバーから以下の手順で進めます：

【Windows】 メニューバーから「View」→「Command Palette」を選択します。

【Mac】

  • 「表示」→「コマンドパレット」を選択
  • または「Command + Shift + P」のショートカットを使用

すると、このような画面が出てきます。 「Java」と入力して、「Java: Create Java Project」を選択します。

押すとこのような一覧が出てくるので、「No build tools」を選択します。

保存先フォルダの作成と選択

フォルダを選択する画面が出てくるので、「educure」を選択してください。 まだ作成していない場合は、新規にフォルダを作成して、「educure」という名前をつけてください。

【Windows】 エクスプローラ画面上部のアドレス欄に%USERPROFILE%と入力して、エンターを押すとご自身のホームフォルダが出てきます。 たいていは初期表示でホーム画面が表示されていると思います。

【Mac】

  • Finderのサイドバーからホームフォルダ（/Users/あなたのユーザー名）を選択
  • または Command + Shift + H でホームフォルダに直接移動

educureフォルダが選択されていることを確認し、右下の「Select the project location」をクリックします。 これで保存先のフォルダの作成、選択ができました。

プロジェクトフォルダの作成

このような入力欄が出てくるので、「1-8」と入力しましょう。 Javaのプロジェクトフォルダが作成されます。ここにこれから作成するJavaファイルを保存していきます。

入力すると、このようなポップアップが出るので、チェックをつけて左の「Yes, I trust the authors」をクリックしてください。

押すとこの画面になります。

これでJavaのプロジェクトの作成は完了です。

Javaファイルの作成

左端の「> src」をクリックし、右クリックして「New File」を選択します。 macOSの場合は、Control + クリックまたは右クリックで「新規ファイル」を選択します。

ファイル名は「HelloWorld.java」としましょう。

Javaファイルの中身の入力

以下のコードをコピーして、VSCodeに貼り付けてください。

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("HelloWorld");
    }
}

貼り付けたら保存します：

【Windows】「Ctrl + S」を押す 【Mac】「Command + S」を押す

プログラムの実行

次にVSCodeのHelloWorldのエディタ部分で右クリックをし、「Run」を選択します。 macOSの場合は、Control + クリックまたは右クリックで「実行」を選択します。

プログラムが実行され、ウィンドウの下半分にターミナルが出てきます。 画像のように「HelloWorld」と表示されていれば、プログラムの実行は成功です。

これで、JavaのプログラムをVSCodeで作成し、実行することができるようになりました。 次の章からこのVSCodeを使ってJavaの学習を進めていきましょう。
